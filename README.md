Qiitanium ([Play Store][google_play_store])
====

![Logo Image][arg_logo]

Qiitaniumは[プログラマのための技術情報共有サービスQiita][qiita_url]の非公式Androidアプリケーションです


### Roadmap to 1.0

- ApiV2によるOAuthユーザ認証
- ユーザ認証による機能拡充（自分の記事一覧、ストックした記事一覧、ストック機能、フォロー機能）
- NotoFontの実験的サポート
